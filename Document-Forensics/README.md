# Digital Forensics Investigation Report

## Objective
Determine whether signatures in a PDF document were digitally manipulated.

## Tools Used
- ExifTool
- pdfinfo
- FotoForensics
- foremost
- binwalk

## Key Findings
- No evidence of signature tampering
- No hidden embedded objects detected
- Metadata confirmed post-creation document modification
- Signatures remained authentic despite document modification
